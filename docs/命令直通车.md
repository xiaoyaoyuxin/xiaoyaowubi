## 输入法设置

| 编码命令     | 功能    |
| --------    | :----:  |
| ocy（常用）      | 输入法常用项设置 |
| omb（码表）      | 输入法码表项设置  |
| 中英文切换  | 右Shift |
| 首选        | 空格    |
| 次选        | ;       |
| 三选        | '       |
| 临时英文    |  ` （Esc下面的键）     |
| 快捷符号引导 | ;       |
| 临时拼音    | z       |
| 编码上屏    | Tab     |
| 清除编码    | Enter       |


## 组合健位

| 功能        | 键位    |
| --------    | :----:  |
| 简繁切换      | Ctrl+Alt+F |
| 繁简切换      | Ctrl+Alt+J |
